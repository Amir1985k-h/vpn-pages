## ✨ ویژگی‌ها

- کانفیگ‌های **تست‌شده و فعال**
- پشتیبانی از پروتکل‌های پرسرعت و امن
- به‌روزرسانی روزانه
- مناسب برای اندروید، iOS، ویندوز، لینوکس و macOS
- سرورهای متنوع (آمریکا، اروپا، آلمان، هلند، سنگاپور و ...)

## 📥 نحوه استفاده

### ۱. نصب اپلیکیشن

**اندروید:**
- [v2rayNG](https://github.com/2dust/v2rayNG)
- [Nekobox](https://github.com/MatsuriDayo/NekoBoxForAndroid)

**iOS / macOS:**
- [Shadowrocket](https://apps.apple.com/app/shadowrocket/id932747118) (پولی)
- [Streisand](https://apps.apple.com/app/streisand/id6450535317) یا [Sing-box](https://github.com/SagerNet/sing-box)

**ویندوز:**
- [Nekoray](https://github.com/MatsuriDayo/NekoRay)
- v2rayN

**لینوکس:**
- v2rayA یا Qv2ray

### ۲. وارد کردن کانفیگ

1. یکی از کانفیگ‌ها را کپی کنید.
2. در اپلیکیشن گزینه **Import** یا **Add Config** را بزنید.
3. کانفیگ را Paste کنید و **Connect** بزنید.

---

## 📂 فایل‌های کانفیگ

- `vmess/` → کانفیگ‌های VMess
- `vless/` → کانفیگ‌های VLESS + Reality
- `trojan/` → کانفیگ‌های Trojan
- `ss/` → کانفیگ‌های Shadowsocks
- `mixed/` → ترکیبی از بهترین‌ها

---

## ⚠️ نکات مهم

- کانفیگ‌ها **موقت** هستند و ممکن است هر لحظه قطع شوند.
- برای استفاده طولانی‌مدت، از سرویس‌های پولی استفاده کنید.
- از **Kill Switch** در اپلیکیشن خود حتماً استفاده کنید.
- هر روز کانفیگ جدید بگیرید.

## 🔄 به‌روزرسانی

کانفیگ‌ها **هر روز** به‌روزرسانی می‌شوند. برای دریافت کانفیگ جدید، دوباره از ربات تلگرام درخواست بگیرید.
